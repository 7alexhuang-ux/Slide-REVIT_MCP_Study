# 內建主題定義

5 種為建築/設計領域設計的簡報主題。

---

## minimal-white — 極簡白

**適合**：正式場合、學術分享、政府機關簡報、多文字內容

| 屬性 | 值 | 說明 |
|------|-----|------|
| 背景 | `#FFFFFF` | 純白 |
| 背景強調 | `#F2F2F2` | 淺灰（程式碼區塊、佔位符）|
| 標題 | `#1A1A1A` | 近黑 |
| 內文 | `#333333` | 深灰 |
| 強調色 | `#007ACC` | 藍色（連結、重點線、項目符號）|
| 副標題 | `#666666` | 中灰 |
| 淡化 | `#999999` | 頁碼、備註 |
| 裝飾 | 底部細藍線 | |

**視覺印象**：乾淨、專業、像建築雜誌的排版。大量留白讓內容自己說話。

---

## architect-dark — 建築深色

**適合**：設計類展示、作品集、夜間場合、投影效果好的場地

| 屬性 | 值 | 說明 |
|------|-----|------|
| 背景 | `#0F0F0F` | 近黑 |
| 背景強調 | `#1E1E1E` | 深灰 |
| 標題 | `#FFFFFF` | 純白 |
| 內文 | `#CCCCCC` | 淺灰 |
| 強調色 | `#B8860B` | 金色（DarkGoldenrod）|
| 副標題 | `#999999` | 中灰 |
| 淡化 | `#666666` | |
| 裝飾 | 左上右下 L 形金色線條 | |

**視覺印象**：高級感、沉穩。金色點綴暗示專業與品味。類似建築競圖的深色背景風格。

---

## blueprint — 藍圖

**適合**：建築/工程/結構主題、技術性較強的內容

| 屬性 | 值 | 說明 |
|------|-----|------|
| 背景 | `#1B2A4A` | 深藍（藍圖底色）|
| 背景強調 | `#243556` | 略淺藍 |
| 標題 | `#FFFFFF` | 純白 |
| 內文 | `#C8D6E5` | 藍灰白 |
| 強調色 | `#5DADE2` | 亮藍 |
| 副標題 | `#85C1E9` | 淺藍 |
| 淡化 | `#7F8C8D` | |
| 裝飾 | 底部 + 右側亮藍線條 | |

**視覺印象**：工程圖、藍曬圖（Blueprint）的聯想。傳達「精密、技術、可信賴」的訊息。

---

## concrete — 清水模

**適合**：材料/結構/施工主題、工業風設計、實驗性建築

| 屬性 | 值 | 說明 |
|------|-----|------|
| 背景 | `#E8E4E1` | 暖灰（混凝土色）|
| 背景強調 | `#D5CEC9` | 深灰 |
| 標題 | `#2C2C2C` | 深色 |
| 內文 | `#4A4A4A` | 中灰 |
| 強調色 | `#8B4513` | 赭色（SaddleBrown）|
| 副標題 | `#6B6B6B` | |
| 淡化 | `#8B8B8B` | |
| 裝飾 | 左側赭色粗色帶 | |

**視覺印象**：清水混凝土（Fair-faced Concrete）的質感。安藤忠雄式的簡潔與力量。赭色暖化了灰色調。

---

## nature — 自然

**適合**：景觀/綠建築/永續發展/環境設計主題

| 屬性 | 值 | 說明 |
|------|-----|------|
| 背景 | `#FAFAF5` | 米白（紙張色）|
| 背景強調 | `#EBE8DF` | 暖米 |
| 標題 | `#2D3436` | 深炭灰 |
| 內文 | `#3D3D3D` | 深灰 |
| 強調色 | `#27AE60` | 草綠 |
| 副標題 | `#636E72` | 灰綠 |
| 淡化 | `#95A5A6` | |
| 裝飾 | 底部綠色線 + 左上角綠色圓點 | |

**視覺印象**：自然、有機、永續。綠色代表生長與環境意識。米色背景比純白更溫暖。

## 主題推薦邏輯

供 AI 在使用者未指定主題時，根據專案內容自動推薦：

| 專案特徵 | 建議主題 | 理由 |
|----------|----------|------|
| BIM / 3D 建模 / 參數化設計 | `architect-dark` | 深色背景凸顯 3D 渲染截圖 |
| 資料分析 / 管理工具 / 報表 | `minimal-white` | 文字與數據為主，需要清晰易讀 |
| 工程計算 / 結構分析 | `blueprint` | 呼應工程圖面的專業感 |
| 材料研究 / 施工記錄 | `concrete` | 材質聯想，工業風 |
| 景觀 / 綠建築 / 永續 | `nature` | 環保綠色主題 |
| 不確定 / 通用 | `minimal-white` | 最安全的選擇 |

## 自訂主題

使用者可以在 frontmatter 中直接覆蓋任何主題屬性：

```yaml
---
theme: "architect-dark"
# 以下欄位在未來版本支援覆蓋
# theme_overrides:
#   accent: "#FF6B6B"
#   font_title: "Noto Sans TC"
---
```

目前版本尚未支援 theme_overrides，如需客製化請直接修改 `generate_pptx.py` 中的 `THEMES` 字典。
